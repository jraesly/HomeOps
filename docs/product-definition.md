# HomeOps Product Definition

## One-Line Description

HomeOps is a mobile-first home operations app for tracking rooms, devices/assets, recurring maintenance, service logs, consumables, documents, warranties, and eventually household knowledge through AI/RAG.

## Core Mental Model

Home
  → Areas / Floors
    → Rooms
      → Devices / Assets
        → Tasks
        → Logs
        → Parts
        → Documents
        → Issues
        → Warranties

## MVP Goal

The MVP proves the maintenance loop:

Create Home
  → Add Room
    → Add Device
      → Add Recurring Task
        → Complete Task
          → Auto-create Maintenance Log
            → Auto-generate Next Due Date
              → Dashboard Updates

## MVP Includes

- Create a home
- Create rooms
- Create devices/assets
- Create recurring maintenance tasks
- Complete maintenance tasks
- Auto-create maintenance logs
- Auto-calculate next due date
- View overdue and due-soon tasks
- View maintenance history for a device

## Not in MVP

- AI/RAG assistant
- OCR
- Full offline sync
- Smart-home integrations
- Contractor sharing
- Household collaboration
- Push notifications
- Web dashboard
- Advanced analytics

## Target First Real Devices

- Water softener
- Whole-house water filter
- Upstairs heat pump
- Main floor heat pump
- Basement/addition heat pump
- Refrigerator
- Dishwasher
- Dryer
- Router/network equipment
- Garden beds
